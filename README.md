# How to backend + postgres

## Set up an npm project

```
npm init -y
```

## Set the database

```
createdb fullstack-pets
touch schema.sql
touch seed.sql
```

### Put stuff in `schema.sql` and in `seed.sql`

- find the nouns, create tables for each
- describe the columns for each kind of thing
